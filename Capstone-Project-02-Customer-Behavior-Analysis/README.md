# Capstone Project 02 – Customer Behavior Analysis

1. Objective

Analyze customer purchasing behavior using the Sample Superstore dataset.

---

2. Dataset

SampleSuperstore.csv

---

3. Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

4. Project Workflow

- Load Dataset
- Customer Segmentation
- Purchase Pattern Analysis
- Retention & Churn Insights
- Business Recommendations
- Conclusion

---

5 Key Insights

- Consumer segment generated the highest sales.
- Purchase patterns vary across product categories.
- Lower-performing customer segments require targeted marketing.
- Customer behavior analysis supports better business decisions.

---

6 Conclusion

The project demonstrates how customer segmentation and purchase pattern analysis can help businesses improve customer engagement and increase sales.

# Capstone Project 03 – Marketing Campaign Analysis

1. Objective

Analyze marketing performance using the Sample Superstore dataset.

---

2. Dataset

SampleSuperstore.csv

---

3. Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

4. Project Workflow

- Load Dataset
- Campaign Metrics Analysis
- ROI Calculation
- Channel Effectiveness
- Business Recommendations
- Conclusion

---

5. Key Insights

- High-performing product categories generate greater sales.
- Estimated ROI reflects overall business profitability.
- Standard Class contributes the highest sales.
- Sales and profit analysis support marketing decisions.

---

6. Conclusion

This project demonstrates how sales and profit data can be used to evaluate marketing performance and improve business strategies.

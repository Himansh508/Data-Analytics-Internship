# Project 04 - Sales Dashboard (Power BI)

1. Overview
This project presents an interactive Sales Dashboard built using Power BI. It provides insights into sales performance, profit, quantity, orders, and discount using KPI cards, charts, and slicers.

2. Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Total Orders KPI
- Average Discount KPI
- Region-wise Sales Analysis
- Category-wise Sales Analysis
- Region Filter
- Category Filter

3. Key Insights
- West region generated the highest sales.
- South region generated the lowest sales.
- Technology category achieved the highest sales.
- Office Supplies generated comparatively lower sales.
- Interactive filters allow dynamic analysis.

4. Tools Used
- Power BI Desktop
- SampleSuperstore Dataset

# 📊 Capstone Project 05 - Global Retail Sales Analytics

## 📌 Project Overview

This project focuses on analyzing a real-world Global Retail Sales dataset using an end-to-end Data Analytics workflow.

The objective of this project is to transform raw retail transaction data into meaningful business insights using SQL, Python, and Power BI.

---

# 🎯 Business Objective

A global retail company wants to understand:

- Which products generate maximum revenue?
- Which products generate maximum profit?
- Which regions perform the best?
- Which markets are underperforming?
- How discounts impact profitability?
- Which customers contribute the most revenue?
- How can the business improve overall performance?

This project answers these business questions through data analysis and visualization.

---

# 📂 Dataset Information

# Project 02: Excel Data Cleaning

1. Company
**ShopSphere** (Fictional E-Commerce Company)

2. Objective

The objective of this project is to clean a raw business dataset and prepare it for further analysis by removing inconsistencies and validating the data.

---

3. Data Cleaning Tasks Performed

- Removed duplicate records.
- Trimmed unnecessary spaces from text fields.
- Standardized text formatting.
- Validated discount values.
- Reviewed dataset for missing and incorrect values.
- Prepared an analysis-ready dataset.

---

4. Outcome

The dataset is now clean, consistent, and ready for further analysis, dashboard creation, SQL querying, Python processing, and Power BI visualization.

---

5. Tools Used

- Microsoft Excel
- Data Validation
- Data Cleaning Techniques

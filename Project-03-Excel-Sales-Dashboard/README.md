# Project 03: Sales Analysis Dashboard (Excel)

1. Objective

Analyze sales data using Microsoft Excel to identify trends, top-performing categories, and business insights through Pivot Tables and a management dashboard.

2. Dataset

Sample Superstore Dataset

3. Tasks Performed

- Cleaned and analyzed sales data
- Created Pivot Tables
- Identified sales trends
- Compared regional performance
- Analyzed category-wise sales and profit
- Built a management-level dashboard
- Used charts for visualization

4. Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity

5. Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

6. Outcome

The dashboard enables management to quickly monitor sales performance, identify high-performing categories, and make data-driven business decisions.

# Project 04: Insight Discovery through SQL

1. 📌 Objective

Analyze business data using SQL to discover meaningful insights by creating relational tables, joining multiple tables, and performing analytical queries.

---

2. 📂 Dataset

Custom **ShopSphere** E-Commerce Database

---

3. 🛠 Tools Used

- MySQL Server 8.0
- MySQL Workbench

---

4. 📋 Tables Created

- Customers
- Products
- Orders
- OrderDetails

---

5. 📖 SQL Concepts Used

- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- PRIMARY KEY
- FOREIGN KEY
- SELECT
- INNER JOIN
- GROUP BY
- SUM()
- Aggregate Functions

---

6. 📊 Business Insights

- Retrieved customer and product information.
- Joined multiple tables using **INNER JOIN**.
- Calculated total sales and profit by product category.
- Calculated total sales by region.
- Built a relational database for business analysis.
